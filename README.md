# agente-trello-dio
🚀 Agente de Automação Trello + Gemini (Projeto DIO)
Este projeto foi desenvolvido como parte do curso na DIO (Digital Innovation One). O objetivo é criar um agente inteligente que gerencia tarefas no Trello utilizando a API do Google Gemini.

🛠️ Tecnologias Utilizadas
Python: Linguagem principal do projeto.

Google Colab: Ambiente de desenvolvimento.

Py-Trello: Biblioteca para integração com a API do Trello.

Google Gemini API: Inteligência artificial para processamento de comandos.

📋 Funcionalidades
Listar Tarefas: O agente lê o quadro do Trello e lista o que precisa ser feito.

Adicionar Tarefas: Cria novos cartões diretamente pelo código.

Mover Tarefas: Atualiza o status (A Fazer -> Em Andamento -> Concluído) automaticamente.

📖 Como usar
Configure suas chaves de API do Trello e do Google Gemini.

Execute as células do notebook no Google Colab.

Use as funções para gerenciar seu quadro de forma automatizada.
